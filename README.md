# 足球比赛预测小程序

这是一个简约现代风格的足球比赛预测微信小程序。用户可以查看每日精选比赛，并获取AI驱动的比赛分析和预测。

## 功能特点

1. **日期选择**
   - 顶部显示日期选择器
   - 支持左右滑动切换日期

2. **精选比赛**
   - 每日展示5场精选比赛
   - 实时获取比赛数据
   - 每场比赛提供AI预测功能

3. **AI预测**
   - 基于DeepSeek的AI分析
   - 提供详细的比赛预测报告

## API接口

1. **足球比赛数据**
   - Host: free-api-live-football-data.p.rapidapi.com
   - 用于获取实时比赛数据

2. **AI预测分析**
   - 使用DeepSeek API进行智能分析
   - 提供比赛胜率预测和详细分析报告
